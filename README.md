# 🛒 Grocery Store App (Flutter)

A mobile grocery application developed using Flutter with basic user authentication and cart functionality.

## 🚀 Features
- 🔐 Login Screen
- 📝 Register Screen
- 🏠 Home Screen (Product Listing)
- 🛍️ Add to Cart
- 📦 Cart Screen
- 🎨 Background UI Design

## 🧰 Technologies Used
- Flutter
- Dart
